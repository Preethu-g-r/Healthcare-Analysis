# Healthcare-Analysis
🏥 Healthcare Analytics Dashboard (Power BI)
An interactive Healthcare Data Analysis Dashboard built using Power BI to analyze hospital performance, patient trends, treatment costs, and diagnostic outcomes.
This dashboard converts raw healthcare data into meaningful insights that support data-driven decision-making in healthcare management.

📊 Dashboard Overview
The dashboard provides a comprehensive view of hospital operations through key performance indicators and visual analytics.
KPIs provide a quick summary of the hospital’s overall performance and patient activity.

Key KPIs
Total Patients: Total number of patients in the dataset.
Total Visits: Total number of hospital visits recorded.
Total Doctors: Number of doctors available.
Average Treatment Cost: Average cost per treatment.
Average Age: Average age of patients.
Total Revenue: Total revenue generated from treatments.
Total Lab Tests: Total lab tests conducted.

📈 Dashboard Visualizations
The dashboard contains multiple charts and visuals to analyze healthcare data effectively.

1.Visits by Quarter
Analyzes patient visit patterns across different quarters and years.
2.Visit ID by Month
Displays monthly patient visit trends to identify peak healthcare demand periods.
3.Cost by Treatment Name
Shows treatment costs for different medications and therapies.
4.Treatment Cost by City
Analyzes treatment expenses across different locations.
5.Top 5 Doctors with Most Patients
Identifies doctors managing the highest number of patients.
6.Test Result Percentage
Shows the distribution of lab results categorized as Normal, Abnormal, and Pending.
7.Patients % by Gender
Provides insights into gender-based patient demographics.
8.Follow-Up Rate
Shows the percentage of patients requiring follow-up visits.

🔍 Key Insights
Patient visits vary across months and quarters.
Certain treatments contribute significantly to healthcare costs.
Doctor workload varies based on patient demand.
Lab test results show normal, abnormal, and pending outcomes.
Around half of the patients require follow-up visits.

🎯 Project Objective
The goal of this project is to demonstrate how data analytics and visualization can improve healthcare management by providing clear insights into patient trends, hospital performance, and treatment outcomes.
